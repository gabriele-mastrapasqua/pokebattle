# poke battle


## Development

install poetry tool.


## install project dependencies
```bash
poetry install
```

## build
```bash
poetry build
```

## run main program
```bash
poetry run main
```

## run tests
```bash
poetry run pytest
```


# Running the project using docker

## start all services
```bash
docker-compose up
```

## tear down
```bash
docker-compose down
```
